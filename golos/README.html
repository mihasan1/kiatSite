EPOLL SYSTEM v3.1 - README

О скрипте:
EPOLL 3.1 - это скрипт для создания неограниченого количества опросов на вашем сайте. Каждый из опросов может иметь до 99-и вариантов ответа. Далее описаны возможности и особенности скрипта скрипта:
- Не использует БД. Вся информация хранится в двух текстовых файлах.
- Нет ограничений на количество созданых опросов.
- Внешний вид формы голосования можно настроить.
- Все операции проводятся через админцентр с паролем.
- Через админцентр можно: Добавлять, Удалять опросы и просматривать их статистику.
- Скрипт выдаёт результаты в виде горизонтальных колонок (разноцветных или серых).
- Скрипт производит правильный подсчёт процентного соотношения голосов.
- Если за какой-либо вариант ответа проголосовало людей меньше 1% (округлённо - 0%), то выводится сотая часть числа процентов (Например: 0,05%).
- Вы можете разместить сразу несколько форм для голосования на одной странице.
- Форма голосования вставляется на страницу при помощи функции include().
- Опции голосования, а также его внешний вид настраиваются на странице с голосованием (перед процедурой include() ).
- Пользоваться скриптом стало ещё легче!

Установка и Проверка:
1) Закачайте все файлы в отдельную папку на вашем сервере и установите для них следующие права доступа:
- add.php (Default)
- admin.php (Default)
- conf.php (Default)
- func.php (Default)
- form.php (Default)
- data.dat (777)
- pollcfg.dat (777)
- password.dat (777)
Создание:
2.1) Откройте в броузере файл admin.php и введите пароль: 1 (Его стоит сменить)
2.2) Зайдите в радел "Создание нового опроса".
2.3) Введите уникальное имя опроса (Например: poll), вопрос и количество вариантов ответа.
2.4) Теперь, если вы всё правильно сделали, вам откроется 2-й шаг создания опроса. Введите варианты ответа и нажмите "Добавить опрос".
2.5) Если написано, что опрос создан - вы создали новый опрос с именем poll.
Размещение:
3.1) Откройте файл, где будет размещён опрос и добавте в него следующий код (меняя значения переменных):
PHP код <?
//Begin Epoll 3.1
$pollname="poll";
$p_path="scripts/epoll/";
$p_submit="Голосовать";
$p_align="center";
$p_width="250";
$p_font="verdana";
$p_txtsize="8pt";
$p_bcolor="#000000";
$p_bgcolor="#E5E5E5";
$p_txtcolor="#000000";
$p_bwidth="1";
$p_days="";
$p_coltype="0";
$p_rwidth="500";
include($p_path."form.php");
//End Epoll 3.1
?>
$pollname - Уникальное имя опроса
$p_path - Путь к папке скрипта
$p_submit - Текст на кнопке submit
$p_align - Выравнивание таблицы с опросом
$p_width - Ширина таблицы
$p_font - Тип шрифта
$p_txtsize - Размер шрифта
$p_bcolor - Цвет границ
$p_bgcolor - Цвет фона
$p_txtcolor - Цвет текста
$p_days - Длительность запрета на повторный голос (в днях), не вводите ничего, если хотите чтобы действовала только проверка на флуд (5 секунд)
$p_coltype - Тип заполнения колонок результатов: 1- разноцветный, 0 - одноцветный (#cccccc) (рекомендую)
$p_rwidth - Ширина таблицы результатов

3.2) Теперь откройте через броузер файл с опросом. Если вы увидели перед собой форму опроса, то вы записали всё правильно.
3.3) Попробуйте проголосовать. После заполнения формы нажмите кнопку "Голосовать". Откроется новое окно с результатами. Если вы увидели надпись: "Ваш голос принят", то скрипт работает нормально.

4) Чтобы вставить другой опрос, нужно разместить тот же код, только поменять имя опроса.
