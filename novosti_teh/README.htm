CuteNews.RU README
 

 Информация о скрипте

 

CuteNews.RU - это мощная и несложная в обращении система публикации новостей, построенная на оригинальной разработке CutePHP Team, использующая в качестве базы данных текстовые файлы. Системные требования для работы скрипта минимальны, а возможности неограниченны! Вы можете использовать CuteNews.RU для ведения новостной ленты, как онлайн-дневник или как настоящую систему управления контентом.

Подробную информацию об использовании скрипта можно почитать на официальном сайте CuteNews.RU!
Все вопросы по использованию скрипта задавайте на форуме для русскоязычных пользователей CuteNews.

 Установка CuteNews.RU

 

Установка CuteNews.RU несложная, следует лишь соблюдать следующие инструкции:

  1. Скачайте последнюю версию скрипта
  2. Создайте директорию на вашем сервере (к примеру: cutenews).
  3. Загрузите содержимое скаченного архива в директорию, созданную в первом шаге.
  4. При помощи FTP-клиента выставьте необходимые права на чтение файлов
    • CHMOD 777 для папки data/ и всех файлов
    • CHMOD 644 для файла index.php
  5. Откройте http://yoursite.com/cutenews/index.php в любом браузере и следуйте инструкциям
  6. После завершения установки скрипта не забудьте удалить файл inc/install.mdu

 Обновление скрипта

 

Внимание! Если у Вас уже установлена более ранняя версия скрипта, и Вы хотите установить новую, сохранив при этом все новости, шаблоны и настройки, следуйте данной инструкции:

  1. Создайте резервную копию всех файлов папки data/
  2. Распакуйте архив с последней версией скрипта во временный каталог
  3. Загрузите все файлы из временного каталога на сервер (кроме файлов папки data/), заменив старые
  4. Откройте Ваш сайт и зайдите в модуль администрирования скрипта
  5. Измените настройка системы (Настройки > Настройка системы)

Таким образом, можно полностью обновить работу скрипта, сохранив все необходимые данные!

 Отображение новостей

 

Главной особенностью скрипта является легкость включения и изменения отображения новостей на страницах сайта. Для включения новостей на страницы вашего сайта необходимо использовать php-функцию include(); и специальные переменные:

  • для отображения определенного количества новостей на странице используйте $number = "x";
  • для отображения новостей из определенной категории используйте $category = "ID,ID,ID"; (ID - номер категории)
  • для отображения новости вместе с заголовками других новостей необходимо использовать $static = TRUE;
Примеры кодов

 

Новости:

<?PHP
 include("path/to/show_news.php");
?>

Новости с другим шаблоном:

<?PHP
 $template = "YOUR_TEMPLATE_NAME";
 include("path/to/show_news.php");
?>

Архивы:

<?PHP
 include("path/to/show_archives.php");
?>

5 последних новостей:

<?PHP
 $number = "5";
 include("path/to/show_news.php");
?>

5 новостей из категории 2:

<?PHP
 $number = "5";
 $category = "2";
 include("path/to/show_news.php");
?>

Все заголовки и последние новости на одной странице:

<?PHP
 $static = TRUE;
 $template = "Headlines";
 include("path/to/show_news.php");

 $number = "1";
 include("path/to/show_news.php");
?>

Помните:

  • Во всех вышеупомянутых примерах, вы должны заменить path/to/... путем к директории, в которую
    вы установили CuteNews и не использовать URL в качестве пути.
    НЕПРАВИЛЬНО: include("http://site.com/cutenews/show_news.php);
    ПРАВИЛЬНО: include("cutenews/show_news.php);                          
  • В большинстве случаев, когда вы хотите включить (include) новости
    и заголовки в одну страницу, используйте $static = TRUE;.
  • Если Вы используете $number, $category, $template или любую другую переменную,
    она должна быть ДО include("show_news.php");
  • Если у вас возникнут какие-либо проблемы, вы всегда можете зайти на наш форум,
    и мы будем рады помочь вам!
